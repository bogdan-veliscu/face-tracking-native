219  /* Get normalized face image.
220  *
221  * This function returns normalized face image with corresponding feature points.
222  * Size of the normalized face in the image is such that interpupillary distance is approximately quarter of the image width.
223  *
224  * Face will be normalized to a varying degree depending on normalization type. For example rotated
225  * face with open mouth will only have its pose straightened with normalization type VS_NORM_POSE while
226  * with addition of VS_NORM_AU normalized face will also have closed mouth.
227  *
228  * Note that the face will always have its pose straightened.
229  *
230  * Types of normalization are:
231  * - VS_NORM_POSE - face translation and rotation are set to zero thereby normalizing the pose
232  * - VS_NORM_SU - parameters describing the face shape (shape units) are set to zero thereby normalizing the face shape
233  * - VS_NORM_AU - parameters describing facial movements (action units) are set to zero, for example open mouth will be closed
234  *
235  * Different types of normalization can be combined with "|" operator, for example VS_NORM_POSE | VS_NORM_SU.
236  *
237  * \b Note: To use this functionality a FaceTrack or FaceDetect license is required.
238  *
239  * @param frame image containing the face to be normalized, must be grayscale
240  * @param normFace image containing the normalized face; it must be allocated before calling the function; face size will depend on this image size
241  * @param face_data FaceData structure containing the information about the face that will be normalized
242  * @param normFDP features points that correspond to the normalized face; coordinates are normalized to 0-1 range
243  * @param norm_type normalization type, a binary combination of VS_NORM_POSE - normalizes pose, VS_NORM_SU - normalizes shape units and VS_NORM_AU - normalizes action units
244  * @param dataPath path to the folder where Face Detector.cfg is located, default values is ""
245  */
246  void getNormalizedFaceImage (
247  VsImage* frame,
248  VsImage* normFace,
249  FaceData* face_data,
250  FDP* normFDP,
251  int norm_type = VS_NORM_POSE,
252  const char* dataPath = ""
253  );
254 
255 };
