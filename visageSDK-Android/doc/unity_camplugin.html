Android Camera Plugin project

The AndroidCameraPlugin is Java plugin used for accessing and handling Android camera. It's use is demonstrated in VisageTrackerUnityDemo and FacialAnimationUnityDemo samples.

Building the project

The files related to Android Camera Plugin project are located in the Samples/Android/AndroidCameraPlugin sub-folder of the visage|SDK for Android folder.

Prerequisites to building the sample project are :

  • Java JDK (version 8 or higher)
  • Android Studio 2.3.2
  • Android SDK (version 4.1.2 API or higher)
  • Unity (version 5.x)
To build the AndroidCameraPlugin, please follow these steps:
  1. Open AndroidCameraPlugin project (located in Samples/Android) into Android Studio.
  2. Navigate to folder AndroidCameraPlugin/app/libs and delete existing file classes.jar
  3. Navigate to directoy where Unity is installed and copy file classes.jar (typically located in C:\Program Files\Unity\Editor\Data\PlaybackEngines?\androidplayer\development\bin)
  4. Paste the file to AndroidCameraPlugin/app/libs
  5. Open Gradle on the right and click "Refresh all Gradle projects" under Gradle projects
  6. Under Gradle projects, go to AndroidCameraPlugin -> :app -> Tasks -> other
  7. Double click on task exportJar and AndroidCameraPlugin.jar will be created in the AndroidCameraPlugin/app/release folder
  8. AndroidCameraPlugin.jar and AndroidManifest.xml should be replaced with new versions in Plugins/Android folder of your Unity project.
    NOTE: Package name in AndroidManifest.xml should match the package name in CameraActivity.jar

Implementation overview

Project is used for handling camera on Android. This is realised as Android activity which is created from C# script in Unity when the application starts. Switching between front and back facing camera is done by calling Android activity's functions from Unity script.
VisageTrackerUnityPlugin provides Java interface to events such as sending frames from camera to tracker and camera parameters changes.
