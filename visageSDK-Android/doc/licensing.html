This section focuses on technical issues such as obtaining and using the license key files. For detailed commercial conditions please contact Visage Technologies.

Applications developed using visage|SDK must be licensed in order to function. Licensing is based on License Key Files. A single License Key File is valid for one or more installations on one or more platforms (operating systems). The License Key File can be opened in a text editor in order to read the licensing information, but it is important not to change the file in any way as it would become invalid.

Applications without a valid license key will still function, but tracking time shall be limited to one minute and appropriate warning messages shall be displayed.

Obtaining the License Key File

For 30-day evaluation, simply send an email request to your Visage Technologies contact person and you will receive a License Key File by email.

If you are purchasing a license, please decide on your Application's Package Name and send it to your Visage Technologies contact person together with your purchase order. In return, you will receive a License Key File by email.

Including the License Key File in your application

For Visage Tracker Demo and Face Detect Demo demo samples, license file should be copied to assets/trackerdata folder.
For Facial Animation Unity and Visage Tracker Unity samples, license file should be copied to Assets\StreamingAssets\Visage Tracker folder.
In your application, you need to call the initialization function in order to let the Application read the License Key File. This is done as follows in your application code: 


// neccessary prototype declaration for licensing
namespace VisageSDK
{
    void initializeLicenseManager(JNIEnv* env, jobject obj, const char *licenseKeyFileName);
}

...

    //initialize licensing
    initializeLicenseManager(env, obj, "absolute_path/name-of-the-license-file.vlc");

where "name-of-the-license-file.vlc" is the name of your License Key File and "absolute_path" is the absolute path to the License Key File (internal or external storage) e.g. "/data/data/com.visagetechnologies.visagetrackerdemo/files/XXX-XXX-XXX-XXX-XXX-XXX-XXX-XXX-XXX-XXX-XXX.vlc".

In the Sample projects the licensing code is already inserted, so you just need to provide the absolute path to the license key file name when you obtain the license key file. Specifically:

In the Visage Tracker Demo sample project, the licensing code is found in the Java_com_visagetechnologies_visagetrackerdemo_TrackerActivity_TrackerInit method, in the AndroidWrapper.cpp file.

In the Face Detect Demo sample project, the licensing code is found in the Java_com_visagetechnologies_facedetectdemo_DetectorActivity_DetectorInit method, in the VisageFeaturesDetectorWrapper.cpp file

In the Visage Tracker Unity Demo and Facial Animation Unity Demo, License File field (script Tracker.cs) needs to be edited in the Inspector as follows:
"license_file_name.vlc"

Additionally, the license key file needs to be added to the list of files to be unzipped.

In the Visage Tracker Unity Demo (script Tracker.cs) and Facial Animation Unity Demo (script VisageTracker.cs) samples, add the license key file name to the Paths Needed field in the method Unzip() as follows:
"license_file_name.vlc"
