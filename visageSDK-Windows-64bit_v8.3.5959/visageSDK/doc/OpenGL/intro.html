Welcome to visage|SDK

visage|SDK integrates a comprehensive range of computer vision technologies related to human faces in an easy-to-use, fully documented Software Development Kit to support a wide area of applications. Main features are face and facial features detection and tracking including 3D head pose, gaze, eye closure, full 3D head model and more, as well as face recognition and face analysis including gender, age and emotion estimation.

Requirements

Supported development environments:

  • Microsoft Visual Studio 2015
  • Microsoft Visual Studio .NET 2015

Note:Most of the provided Sample projects can not be built in Express or Community versions of Visual Studio due to lack of MFC support. The visage|SDK libraries do not use MFC but many of the Sample projects do.

Additional features required for Visual Studio 2015 installation:

  • Common Tools for Visual C++ 2015
  • Microsoft Foundation Classes for C++
  • Microsoft .NET Framework 4.0

Documentation

Visage|SDK documentation consists of this user manual and additional reference manuals for specific purposes (sample projects, tracker configuration etc.). These additional manuals are accessed through links in this document, the References section, or directly in the doc folder.

The menu on the left allows navigation through the documentation.

Getting started

visage|SDK includes sample projects with full source code demonstrating how to use the API. It is recommended to start by running the samples, described in the Samples section and available in pre-compiled form in the bin folder.
The API is given in the API section and contains full documentation of all classes organised by topic.
