Detailed Description

Implements simple high-level Objective-C interface around VisageSDK::VisageFeaturesDetector functionallity.

This makes connecting the OS X app GUI with visage|SDK features detector functionaly easier as primary programming language used in OS X apps is Objective-C while primary programming language used in visage|SDK is C++. The class is a subclass of NSObject and it uses Objective-C++ for implementation details.
