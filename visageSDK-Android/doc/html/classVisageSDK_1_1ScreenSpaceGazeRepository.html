Detailed Description

Data repository holding ScreenSpaceGazeData objects in offline screen space gaze tracking mode.

The repository is used provide ScreenSpaceGazeData objects holding calibration data for the tracked video sequence to the tracker, as well as to obtain the results. Each ScreenSpaceGazeData object stored in the repository holds the calibration data for a single frame. Starting calibration index should be equal or larger than 1.
