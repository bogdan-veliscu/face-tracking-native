visage|SDK API

visage|SDK provides two APIs: the native C++ API, and the C# API implemented through a wrapper.

The following sections provide an overview of the APIs and give links to most important classes, as well as to sample projects demonstrating particular functions.

C++ API

The C++ API contains vision-based algorithms for detecting and tracking facial features in images/videos as well as face analysis (gender, age and emotion estimation) and face recognition. It is implemented in three separate libraries in order to optimize the size of specific applications (listed with their dependencies):

  • libVisageVision64.dll - face detection, alignment and face tracking, requires:
    • libopenblas.dll
      1. libgfortran-3.dll
      2. libquadmath-0.dll
      3. libgcc_s_seh-1.dll
  • libVisageGaze64.dll - screen space gaze tracking, requires:
    • libVisageVision64.dll
  • libVisageAnalyser64.dll - face analysis and recognition, requires:
    • libVisageVision64.dll

Facial feature tracking

Visage tracker tracks multiple faces and facial features in video sequences and outputs 3D head pose, facial expressions, gaze information, eye closure, facial feature points and full 3D face model. Tracker is fully configurable in terms of performance, quality, as well as other options. Common configurations are delivered. Details about configuring the tracker can be found in the VisageTracker Configuration Manual.

Face tracking is demonstrated in FaceTracker2, VisageTrackerUnity, FacialAnimationUnity and SimpleTracker sample projects.

Main classes

  • VisageTracker: Tracks multiple faces and facial features.
  • FaceData: Used to return tracking results from the tracker.
  • VisageGazeTracker: Screen-space gaze position tracking.

Facial feature detection

The class VisageFeaturesDetector detects faces and facial features in input images. The results are, for each detected face, the 3D head pose, gaze direction, eye closure, the coordinates of facial feature points, e.g. chin tip, nose tip, lip corners etc. and 3D face model fitted to the face. The results are returned in one or more FaceData objects, one for each detected face.

The Face Detector sample demonstrates facial feature detection.

Main classes

  • VisageFeaturesDetector: Detects faces and facial features in still images.
  • FaceData: Used to return detection results from detector.

Face analysis

The class VisageFaceAnalyser contains face analysis algorithms capable of estimating gender, age and emotion from frontal facial images (yaw between -20 and 20 degrees).

The Face Detector sample demonstrates gender, age and emotion estimation.

Main classes

  • VisageFaceAnalyser: Face analyser capable of estimating gender and emotion from frontal facial images (yaw between -20 and 20 degrees). For age estimation it returns estimated age, for gender estimation it returns estimated gender and for emotions estimation it returns the probability of each of estimated facial emotions: anger, disgust, fear, happiness, sadness, surprise and neutral.

Face recognition

The class VisageFaceRecognition contains a face recognition algorithm capable of measuring similarity between human faces and recognizing a person's identity from frontal facial image (yaw angle approximately from -20 to 20 degrees) by comparing it to faces previously stored in a gallery. It returns name of the person with the most similar face found in the current gallery.

The Face Detector sample demonstrates face recognition.

Main classes

  • VisageFaceRecognition: Measures similarity between human faces and recognizes a person's identity from frontal facial images (yaw angle approximately from -20 to 20 degrees).

C# API

The C# API provides access to vision-based algorithms for detecting and tracking facial features in images/videos as well as face analysis (gender, age and emotion estimation) and face recognition in C#. It is implemented in libVisageCSWrapper.dll library, and requires all basic C++ libraries for running (libVisageVision, libVisageGaze and libVisageAnalyser).

  • libVisageCSWrapper.dll - requires:
    • libVisageVision64.dll
      • libopenblas.dll
        1. libgfortran-3.dll
        2. libquadmath-0.dll
        3. libgcc_s_seh-1.dll
    • libVisageGaze64.dll
    • libVisageAnalyser64.dll
