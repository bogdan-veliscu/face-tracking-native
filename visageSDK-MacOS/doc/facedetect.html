Face Detect Demo sample project

The Face Detect Demo sample project demonstrates facial features detection based on the VisageSDK::FacialFeaturesDetector class.

Building and running the project

The FaceDetectDemo sample application is located in bin subfolder of visage|SDK for Mac OS X folder. To run the sample application double click on FaceDetectDemo file.

The FaceDetectDemo sample project is located in the Samples/MacOSX/FaceDetect subfolder of the visage|SDK for OS X folder.

Prerequisites to building the sample project are:

  • Xcode (versions 7.x+)

Project is opened in Xcode by double-clicking on the FaceDetectDemo.xcodeproj project file. To build the project select Product->Build from menu bar. The project sample app can be run by choosing Run option in Xcode.

Using the sample app

The sample app consists of a main window and a menu. Image is selected by File->Open in the menu. Results of face detection are displayed in the window. Sample images are provided in data/images sub-folder of visage|SDK for Mac OS X folder.

NOTE: Without the valid license appropriate message will appear and detection results will be shown with a 2 seconds delay.


Implementation overview

The visage|SDK API is written in C++ and since development on OS X is done in Objective-C, an interface between them must be implemented. In this sample project this is done through VisageFeaturesDetectorWrapper Objective-C class that exposes Objective-C methods for using the detector, while the implementation of those methods is done in Objective-C++, a combination of Objective-C and C++.

The AppDelegate (Objective-C class) connects with GUI (that is defined in MainWindow.xib file) and implements actions that calls to methods from VisageFeaturesDetectorWrapper class to detect and display facial features.

The specific classes and methods that demonstrate visage|SDK are:

  • VisageFeaturesDetectorWrapper: Provides the Objective-C interface to main face detector functions.
  • VisageFeaturesDetectorWrapper::drawFeatures: Method that is used to detect and display the detected features.

Other important classes are:

  • AppDelegate: Implements the app delegate.

More details about each class is available in documentation for specific class.

Users are free to modify the sample project to test the functionalities of the Facial Features Detector. The sample project also serves as a starting point when creating new projects that will use visage|SDK.
