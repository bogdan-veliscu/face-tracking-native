Detailed Description

AndroidCameraCapture demonstrates use of raw camera image input to track from Android camera.

GrabFrame method will be periodically called to get new frame. For inputing new frame, WriteFrame should be used. This method expects frame in Android camera NV21 format (YUV420sp). YUV420sp to RGB converting, rotation and flipping is done in GrabFrame.
