Visage Tracker Unity Plugin (C# wrapper) project

The VisageTrackerUnityPlugin is a wrapper that exposes some of VisageTracker's functionality for use in managed programming languages such as C# and Java. A prebuilt version of the wrapper is provided in libs/libVisageTrackerUnityPlugin.so. It's use is demonstrated in VisageTrackerUnityDemo and FacialAnimationUnityDemo samples where a C# script is used to call functions exposed in this library. This project is designed to be integrated in other applications and cannot be run on its own.

Building the project

The files related to Visage Tracker Unity Plugin project are located in the Samples/Android/VisageTrackerUnityPlugin sub-folder of the visage|SDK for Android folder.

Prerequisites to building the sample project are :

  • Android NDK (revision r10e)
To build the VisageTrackerUnityPlugin the Android NDK's "ndk-build.cmd" (or equivalent on linux) must be executed from inside the Samples/Android/VisageTrackerUnityPlugin folder. The quickest way to do this is to open the command line and execute the following commands on Windows (or equivalent on linux):
  • cd [path to VisageTrackerUnityPlugin]
  • [path to Android NDK]\ndk-build.cmd
For example:
  • cd c:\VisageSDK\Samples\Android\VisageTrackerUnityPlugin
  • c:\android-ndk-r9d\ndk-build.cmd
The NDK will build the library in the libs subfolder of the VisageTrackerUnityPlugin project folder. The build will include all dependent libraries.

Implementation overview

The project consists Android make scripts and C++ source code and header files. It demonstrates how to use some of Visage Tracker's properties such as translation, rotation and Action Units and wraps some of Visage Tracker's functions such as initiating and stopping tracking. More functionality can be exposed in the same manner. Refer to the API documentation for more information on how to access various properties and functionalities of the Visage Tracker.
VisageTrackerUnityPlugin is also used as Java interface since camera handling and sending frames to tracker is realised in Java plugin.
