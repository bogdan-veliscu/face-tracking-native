visage|SDK samples

Samples are ready-to-run projects demonstrating the use of visage|SDK. They are intended to provide an easy start for developers using the SDK, and can also be used as a basis for building own applications. Sample projects are provided with full source code.

Samples are organized into three sections: C++ samples (based on the C++ API), C# samples (based on the C# API) and samples showing integration with the Unity3D engine.

Sample projects are described in the tables below. The documentation for each sample project provides details on running and using the sample, as well as implementation details.

C++ Samples

SAMPLE NAME OVERVIEW
Face Tracker 2 Demonstrates facial features tracking on multiple faces from camera, video file or image and retrieving/displaying tracking results.
Simple Tracker A minimalistic console application demonstrating facial features tracking from camera or video file, aimed at easy understanding of the code.
Face Detector Demonstrates facial features detection in still images; detects multiple faces. Demonstrates face recognition and age, gender and emotion estimation in still images; face recognition, age and gender estimation working on multiple faces. Emotion estimation works only on the single largest detected face.
Gaze Tracker Demonstrates online mode of screen space gaze estimation (from the camera).

Most sample applications are based on MFC and may use OpenGL for 3D rendering.

C# Samples

SAMPLE NAME OVERVIEW
Showcase Demo Demonstrates facial features tracking on multiple faces from camera, video file or image and retrieving/displaying tracking results. Also demonstrates facial features detection in still images; detects multiple faces. Demonstrates face recognition and age, gender and emotion estimation on multiple faces.

Unity3D Samples

SAMPLE NAME OVERVIEW
VisageTracker Unity Plugin Plugin that exposes some of VisageTracker's functionality for use in managed programming languages such as C# and Java; used for integration with Unity3D.
VisageTracker Unity Integration of visage|SDK with Unity game engine; displaying a virtual object (glasses) on the face and full 3D model of the face (face mask effect).
Facial Animation Unity Integration of visage|SDK with Unity game engine; using action unit driven blendshapes.

Source and data files

Sample projects can be found in the Samples\OpenGL folder. Subfolder data holds data files used by the sample applications (models, animations etc.), while sample source code and project files can be found in source and build subfolders, respectively. visage|SDK package comes with pre-built versions of more attractive sample applications (executables with "DEMO_" prefix). These pre-built samples are found in the bin64 folder. If you wish to build a sample yourself, you can use the Visual Studio 2015 solution (Samples\OpenGL\build\msvc140\Samples.sln).
